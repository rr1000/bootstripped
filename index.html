Welcome To BootStripped

"a minimalist version of twitters 2.0 bootstrap foundation"

The basis of BootStripped is creating a css foundation, much like Bootstrap, but without most of the obvious styling that comes with bootstrap. BootStripped gives architecture rather than experience in terms of site layout.

You'll notice that BootStripped utilizes multiple css classes within single elements. By using object oriented styling. This allows one to apply styles fast, effective, and carefree. Many classes are prebuilt and can be added to any elements in the HTML.
